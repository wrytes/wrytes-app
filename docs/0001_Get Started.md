# Get Started

## What is Wrytes?

Wrytes AG is a Swiss R&D company based in Zug, Switzerland, specialising in software development for Distributed Ledger Technologies and AI. This platform is the internal tooling and research environment used to interact with DeFi protocols, manage wallets, monitor positions, and run development experiments.

## Prerequisites

Before using the platform, ensure you have:

- A modern browser with JavaScript enabled
- A Web3 wallet — MetaMask, WalletConnect-compatible wallet, Coinbase Wallet, or Safe
- A Telegram account linked to your wallet (required for authentication)
- Access to Ethereum Mainnet or Base L2

## How to Access

1. Navigate to the app URL
2. Click **Connect Wallet** in the top-right header
3. Complete the authentication flow (see [Authentication](/docs/0002_Authentication))
4. You will land on the Dashboard once approved
